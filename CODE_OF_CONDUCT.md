# Code of Conduct

## Our Pledge

We pledge to make participation in this project harassment-free for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being clear about problems.
- Keep feedback focused on the work.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Sustained disruption of project discussion.

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code.
